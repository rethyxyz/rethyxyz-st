# rethyxyz-st
My build of st. Minimally patched. Contains transparency, and scrollback patches.

Keybindings are as follows:
- **Alt-minus/equal**: Font size +1/-1
- **Alt-ShiftMask-Plus**: Reset font size
- **Alt-j/k**: Scroll back/up
- **Ctrl-Shift-Y**: Paste
- **Ctrl-Shift-c/v**: Copy/paste
- **Shift-Insert**: Paste
