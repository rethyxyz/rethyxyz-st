# rethyxyz-st
My build of st. Minimally patched. Contains transparency, and scrollback patches.

Keybindings are as follows:
- **Alt-Shift-K/J**: Font size +1/-1
- **Alt-ShiftMask-Home**: Reset font size
- **Alt-k/j**: Scroll back/up
- **Ctrl-Shift-Y**: Paste
- **Ctrl-Shift-c/v**: Copy/paste
- **Shift-Insert**: Paste
